Программа с GUI интерфейсом запускается из папки Database командой:

python gui.py

****************
*НАЧАЛО РАБОТЫ:*
****************
В начале работы нужно либо загрузить базу данных из csv файла, либо создать новую.


********************************
*ЗАГРУЗКА БАЗЫ ДАННЫХ ИЗ ФАЙЛА:*
********************************
Чтобы загрузить базу данных из csv файла, кликните на меню File, затем Load и в открывшемся окне выберите csv файл.


***********************
*СОЗДАНИЕ БАЗЫ ДАННЫХ:*
***********************
Чтобы создать базу данных, кликните на меню File, затем Create и в открывшемся окне задайте число полей базы, их названия и значения.


*****************
*РЕДАКТИРОВАНИЕ:*
*****************
Чтобы отредактировать ячейку, дважды кликните по ней, либо выберите меню Edit, затем Edit row, затем задайте строку, столбец и значение ячейки.


***********
*УДАЛЕНИЕ:*
***********
Чтобы удалить строку, выделите ее в таблице, кликнув по заголовку строки, затем кликните меню Edit, затем Delete Row. Будет удалена текущая строка.


*************
*ДОБАВЛЕНИЕ:*
*************
Чтобы добавить строку, кликните меню Edit, затем Add row,затем в открывшемся окне задайте значения строки. В базу данных будет добавлена строка.


********
*ПОИСК:*
********
Чтобы выполнить поиск по столбцу, выделите столбец, кликнув по заголовку столбца, затем кликните меню More, затем Find row, затем в открывшемся окне введите значение для поиска. Найденные строки будут выделены в таблице.


************************
*ФОРМАТ ВХОДНЫХ ДАННЫХ:*
************************
Формат csv файла:
поля разделены запятыми, первая строка -- строка с названиями поле, первое поле слева -- уникальный целочисленный числовой идентификатор, начиная с 0.
Пример формата csv файла:
,заголовок1,заголовок2,заголовок3,заголовок4
0,значение,значение,значение,значение,
1,значение,значение,значение,значение,
2,значение,значение,значение,значение,
